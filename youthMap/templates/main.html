The Details

YOUTH MAP의 지도 사용 가이드

해당 지도는 카카오 맵 API를 기반으로 하였습니다

지도의 시작 위치

군산대학교를 초기 지도 시작 위치로 설정하였습니다

시설에 대한 마커

각 시설의 위치에 마커를 찍어, 해당 위치를 직관적으로 알 수 있게 하였습니다

마커 클릭 시 시설 정보 출력

마커를 클릭하면 해당 시설에 대한 간략한 정보를 출력합니다 (기관명, 주소, 전화번호)

자유로운 지도의 확대 및 축소

정적지도가 아닌 동적 지도를 사용하여 지도의 확대 및 축소가 가능하게 하였습니다

시설 찜 하기

가고 싶은 시설을 찜해서 개인별 찜목록을 만들 수 있습니다

로그인 / 회원가입

개인별 찜목록을 만들기 위한 로그인 / 회원가입 기능입니다. 로그인 상태일때만 찜을 누를 수 있고, 찜 목록 확인이 가능합니다

사용 기술

오라클 데이터베이스

Oracle Database 21c 버전을 사용하였습니다. 파이썬 import oracledb를 통해 DB 연동을 진행하였습니다.

  • Learn More

파이썬 플라스크

서버 연동을 위해 플라스크 라이브러리를 사용하기로 했습니다. 파이썬 라이브러리이기에 서버 언어는 파이썬을 사용하였습니다.

  • Learn More

제이쿼리, 자바스크립트

파이썬의 데이터를 받아오기 위해 JQuery의 ajax를 이용하였으며, 기타 기능 구현을 위해 자바스크립트를 사용하였습니다.
